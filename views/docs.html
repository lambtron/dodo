HTTP API documentation

You can also add people to your Dodo list by sending an HTTP request. Note that you must authorize your Twitter account with Dodo in order to use this API.


Endpoint

https://dodo-twitter.herokuapp.com/api/dodo


Sample request

The userId is your Twitter userId. The dodoId is the Twitter userId of the account you want to add to your Dodo list. The content-type must be application/x-www-form-urlencoded. 

{
  userId: 123413414
  dodoId: 32534234145
}

Sample response

OK
