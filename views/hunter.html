Hunter

This service will listen to the firehose of tweets and, upon finding a tweet that matches a filter that you set, will follow, mute, and add the user to your Dodo list.

Both of these services require you to register a Twitter application and set the Twitter Consumer Key and Secret.
- hunter-scan: finds matching tweets and adds user to MongoDB
- hunter-follow: scheduled cron job to follow, mute, and add the user to your Dodo list
